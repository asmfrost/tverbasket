# tverbasket
> Тестовое задание для ОАО «НПО РусБИТех»

## Установка

```bash
$ npm install -g grunt-cli
```

```bash
$ git clone git@github.com:asmfrost/tverbasket.git
$ cd tverbasket
$ npm install
$ grunt bower:install
```

## Компиляция

1. `grunt build` - версия для отладки
2. `grunt release` - версия для распространения
